# youtube_scapping
# 📺 YouTube Video Scraper (with Selenium + BeautifulSoup)

This project scrapes YouTube video details from a list of video URLs using Selenium and BeautifulSoup.  
It extracts information like:

- 🎯 Video Title
- 👀 View Count
- 👍 Likes
- 🕒 Upload Date
- 📝 Description
